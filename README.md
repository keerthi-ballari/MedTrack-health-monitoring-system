MedTrack is a cloud-enabled healthcare management system built using Flask.

## Features
- User login and registration
- Doctor dashboard
- Patient dashboard
- Appointment booking
- Appointment viewing

## Tech Stack
- Python
- Flask
- HTML
- CSS
